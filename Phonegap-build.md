#iOS phonegap build

+ Install phonegap

    `$ sudo npm install -g phonegap`

+ Add build target

    `$ phonegap platform add ios`

+ Add the plugins

    `$ phonegap plugin add git://git.apache.org/cordova-plugin-inappbrowser.git`

+ Build

    `$ phonegap build`

+ Emulate

    `$ phonegap emulate ios`
